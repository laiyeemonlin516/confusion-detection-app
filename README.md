# User-Specific Character Confusion Detection

This is a prototype application for handwritten character confusion detection.

## Current Flow
Input target character  
→ confusion pair probabilities  
→ gap calculation  
→ threshold-based highlight

## Features
- Target character input
- Confusion pair probability calculation
- Gap calculation
- Threshold-based highlight
- Streamlit-based web application

## Note
This version uses simulated probabilities instead of a trained model.
A real image-based model can be integrated later.

## Run
streamlit run app.py
